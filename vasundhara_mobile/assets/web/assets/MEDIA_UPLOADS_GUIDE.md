# Vasundhara Palace - Master Media Upload Guide

This document is your definitive reference for upgrading and replacing all photos and videos on the Vasundhara Palace website. 

To replace any visual on the website, **simply name your new file exactly as it is written in this table**, drop it into this `assets/` folder, and it will immediately update on the live site. 

*Note: For the best performance, try to keep video files under 15MB.*

### How to use this sheet in the future:
Whenever you shoot new footage or have a professional photoshoot of a new event, consult this table. If you want to replace the Party Hall video, just name your new video `party-vid-1.mp4`, paste it into this folder (replacing the old one), and refresh the website. The system will handle all the cropping, sizing, and styling automatically!
